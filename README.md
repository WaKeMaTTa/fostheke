# Fostheke

## Etymology

From *φώς* (fós, “light”) + *-θήκη* (-thḗkē, “box, chest”).

Inspired from the Ancient Greek word: [*βιβλιοθήκη* (biblíonthḗkē, “library”)][ancient-greek-bibliontheke].
